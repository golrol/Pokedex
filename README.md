# Pokedex
## Running instructions ##
1. Download repo
```
git clone https://github.com/golrol/Pokedex
```
2. Install required modules
```
npm install
```
3. Run the server
```
npm start
```
5. Enter following link in browser:
```
localhost:3000
```
(if the port is already in use, change the port number in server.js)
